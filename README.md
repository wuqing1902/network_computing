# Network Computing Portfolio

This repository documents my hands-on learning journey in computer networking, including network configuration, protocol analysis, troubleshooting exercises, and lab simulations.

The goal of this project is to develop practical networking skills and demonstrate an understanding of core network technologies used in real-world IT environments.

This repository contains practical networking labs, packet analysis exercises, and configuration examples using tools such as Cisco Packet Tracer, Wireshark, and Linux networking utilities.

<br><br>

## Objectives

- Develop practical networking skills through lab environments
- Understand fundamental networking protocols and concepts
- Practice network configuration and troubleshooting
- Analyze network traffic and packet behavior
- Document networking experiments and learning outcomes

<br><br>

## Topics Covered

### Network Fundamentals
- OSI Model
- TCP/IP Model
- IP Addressing
- Subnetting
- ARP and ICMP

### Network Configuration
- Router configuration
- Switch configuration
- VLAN configuration
- Static routing
- DHCP configuration

### Network Analysis
- Packet capture using Wireshark
- Protocol analysis (TCP, UDP, DNS, HTTP)
- ARP investigation
- Network troubleshooting techniques

### Networking Tools
Hands-on experience with networking tools including:

- Cisco Packet Tracer
- Wireshark
- Linux networking commands
- Netstat
- Ping and Traceroute
